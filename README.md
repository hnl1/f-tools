# 剪切板链接管理器

## 🌟 功能

- 读取剪切板链接 `Ctrl + V`
    - 自动去除不可见字符
- 写入剪切板
    - 链接类型 `Ctrl + Enter`
    - Markdown 类型 `Ctrl + Shift + Enter`

## 🚀 使用

- 在线使用[剪切板链接管理器](https://hnl1.github.io/copy-link/)
- 下载到本地，浏览器打开 [index.html](index.html)
- 下载到本地，启动任意 web 服务，例如 Python 的 `python3 -m http.server 8000`
