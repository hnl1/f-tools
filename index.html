剪切板链接管理器

把名称和地址打包成富文本 / Markdown 链接，一键复制到剪切板。

🎬

视频对比

拖入多个视频文件并排播放，可同步播放暂停，方便对比效果差异。

→
🖼️

图片对比

多张图片平铺，鼠标移动时同步放大相同区域，逐像素查看细节。

→
📄

PDF 对比

拖入多个 PDF 横向平铺，上下滚动时所有列按比例或按页同步。

→
🧾

PDF 转图片版

把 PDF 每页渲染成图片后重新封装，并可直接跳转到 PDF 对比查看差异。

→
🏷️

拖入 PNG，自动解析尺寸、色彩、Gamma、ICC、tEXt 等元数据与 chunk 列表。
